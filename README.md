# Soubory ke stazeni

Jednoducha verze pro Vercel: `index.html`, `style.css`, `script.js` a serverless skripty v `api`.

Na Vercelu vytvor v projektu Storage -> Blob. Vercel prida `BLOB_READ_WRITE_TOKEN`.

Doporuceno: nastav v Environment Variables `UPLOAD_PASSWORD`. Pak se bez hesla neda nahravat, ale stahovani zustane verejne.
